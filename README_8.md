## Описание программы

`Random Task Generator` — это простое и удобное приложение с графическим интерфейсом (GUI) для тех, кто не знает, за что взяться, или хочет внести элемент случайности в свою рутину. Программа позволяет создать свой пул задач (учёба, работа, спорт и др.) и случайным образом выбирать одну из них для выполнения.

### Основные возможности:

1.  **Генератор задач:** Нажатие одной кнопки выбирает случайную задачу из общего списка или из определенной категории.
2.  **Пользовательский список:** Вы можете легко добавлять собственные задачи, указывая их категорию. Защита от пустого ввода не позволит добавить некорректную задачу.
3.  **Фильтрация:** Встроенный фильтр позволяет генерировать и просматривать историю задач по конкретным категориям (например, если вы сейчас на работе, выберите фильтр "Работа").
4.  **Сохранение данных:** Программа автоматически сохраняет как пул добавленных вами задач (`tasks_pool.json`), так и историю выпавших задач (`task_history.json`). Ничего не потеряется после закрытия приложения.

## Инструкция по запуску

Приложение использует только стандартные библиотеки Python, сторонние зависимости не требуются.

1.  Убедитесь, что у вас установлен **Python 3**.
2.  Скачайте файлы проекта в одну папку.
3.  Откройте терминал (командную строку) в этой папке.
4.  Запустите программу командой:
    ```bash
    python random_task_generator.py
    ```

## Примеры использования (Тесты)

*   **Тест 1: Генерация базовой задачи.**
    Сразу после запуска (фильтр "Все") нажмите "Сгенерировать задачу". В центре экрана появится текст задачи, а внизу в таблице — новая запись об этом.
*   **Тест 2: Добавление новой задачи.**
    В верхнем блоке введите "Помыть посуду", выберите категорию "Другое". Нажмите "Добавить". Появится сообщение об успехе. Выберите фильтр "Другое" и нажмите генерацию — вам выпадет именно эта задача.
*   **Тест 3: Защита от пустого ввода.**
    Оставьте поле ввода задачи пустым и нажмите "Добавить". Появится окно с ошибкой "Текст задачи не может быть пустым!".
*   **Тест 4: Работа фильтра в истории.**
    Сгенерируйте несколько задач из категории "Спорт" и несколько из "Работа". Затем поменяйте фильтр (выпадающий список) на "Спорт". Обратите внимание, что таблица истории обновилась и показывает только спортивные задачи.
*   **Тест 5: Сохранение данных.**
    Добавьте уникальную задачу, сгенерируйте её, затем закройте программу. Запустите заново. Ваша задача останется в списке генерации, а в истории сохранится запись о её выполнении.
